How to Configure Your AIPortal

To customize the AIPortal according to your preferences, you need to adjust the config.ini file. This file controls various aspects of the portal, such as the top menu items, their URLs, and the site title. Here's a quick guide:

  1. Site Title: To set the site title, find the [titel] section and modify the titel_name value. For example: titel_name = "Your Site Title".
  2. Top Menu: Under the [menu] section, you can define each menu item. Specify the URL, name, and position for each item. For example:
    menu_1_url = "./home.html"
    menu_1_name = "Home"
    menu_1_position = 1
    Repeat for each menu item you wish to add. If a menu item should have a dropdown, use a "#" for its URL and define sub-items similarly, with a unique identifier.
  3. Adding New Categories to the LoraStore: Within the lorastore directory, there's a folder named Database-Blank. You can copy or duplicate this folder to create a new category in the LoraStore menu. To include this new category in the menu, refer to the instructions above for adding a new menu item under the [menu] section of the config.ini file. This allows you to expand the content available on your AIPortal easily.

Remember to save your changes and upload the updated config.ini file to your server. Refresh your AIPortal to see the changes in effect.
